### 4. Tópicos Discutidos

#### 4.1 Alinhamento
*   O grupo realizou uma sessão de ideias abertas para o direcionamento da entrega 2.
*   Foi discutido o que deveria ser entregue, e a forma como o grupo se organizaria para esse fim.

#### 4.2 Planejamento
*   O grupo fez um planejamento visando atacar os 3 focos da entrega, em 3 dias distintos (segunda, terça e quarta).
*   Na segunda-feira, os membros disponíveis se reuniriam para desenvolver o artefato para o foco *Modelagem Estática*, que seria um diagrama de classes.
*   Na terça-feira, os membros disponíveis se reuniriam para desenvolver o artefato para o foco *Modelagem Dinâmica*, que seria um diagrama de estados.
*   Na quarta-feira, os membros disponíveis se reuniriam para desenvolver o artefato para o foco *Modelagem Organizacional*, que seria um diagrama de pacotes.
*   Os artefatos extras seriam desenvolvidos em tempo oportuno pelos integrantes disponíveis. Haveria intenção de desenvolver um *backlog do produto*, um *diagrama de sequências* e um *diagrama de implantação* como artefatos extras.
